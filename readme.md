# Notifier Desktop Client

Aplikacja desktopowa stworzona w **Electron** do wyświetlania powiadomień z bazy danych MySQL.  
Pozwala na pobieranie powiadomień, pokazanie ich w GUI i łatwe zarządzanie użytkownikami.

---

## 🛠 Wymagania

- [Node.js](https://nodejs.org/) (w wersji LTS, np. 18.x lub nowsza)  
- [npm](https://www.npmjs.com/) (instalowane razem z Node.js)  
- Baza danych **MySQL** z tabelami `users` i `notifications`  
- Pakiet **Electron** (zainstalowany lokalnie w projekcie)
